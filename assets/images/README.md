# Image Assets Directory

Place your images in this directory:

## Required Images

### Profile Images
- `headshot.jpg` - Your professional headshot (recommended: 400x400px, square format)

### Project Screenshots
Name your project images descriptively:
- `project-dribble-ai.jpg` - Dribble.AI project screenshot
- `project-finlens.jpg` - FinLens project screenshot
- `project-docusense.jpg` - DocuSense.AI project screenshot
- `project-agrotalk.jpg` - AgroTalk project screenshot
- `project-semantic-clustering.jpg` - Semantic Clustering project screenshot
- `project-d2d.jpg` - Drawing-to-Data project screenshot

## Image Guidelines

- **Format**: JPEG for photos, PNG for graphics with transparency
- **Size**: Optimize for web (keep under 500KB per image)
- **Dimensions**: 
  - Headshot: 400x400px (square)
  - Project screenshots: 1200x800px or 16:9 aspect ratio
- **Quality**: 80-85% JPEG quality is usually sufficient

## Optimization Tools

- [TinyPNG](https://tinypng.com/) - Online image compression
- [Squoosh](https://squoosh.app/) - Advanced image optimization
- ImageOptim (Mac) - Desktop app for optimization

## After Adding Images

Update the corresponding component files:
- `src/components/Hero.jsx` - Update headshot image path
- `src/components/Projects.jsx` - Update project image paths in the projects array
